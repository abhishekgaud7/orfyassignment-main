# Productr - Figma Design Implementation

This is a React Frontend implementation of the "Productr" design (Figma).

## Features

- **Authentication Flow**: Login and OTP screens with simulated verification.
- **Dashboard**: Responsive sidebar layout.
- **Product Management**:
  - List products (Grid view).
  - Add new products.
  - Edit existing products.
  - Delete products with confirmation.
  - Publish/Unpublish toggle.

## Tech Stack

- React (Vite)
- CSS Modules (Vanilla implementations)
- JS (ES6+)

## Setup

1. `npm install`
2. `npm run dev`
